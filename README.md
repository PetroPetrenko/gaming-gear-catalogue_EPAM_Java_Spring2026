# Gaming Gear Catalogue

## Description

Gaming Gear Catalogue is a responsive catalogue web page created using HTML5, CSS3 and JavaScript.

## Features

- Product catalogue
- Product search
- Product sorting
- Dark mode
- Responsive layout
- Add to cart interaction

## Technologies

- HTML5
- CSS3
- JavaScript

## Run Project

1. Download the project
2. Open index.html in browser
