# Functional Requirements

- User can open catalogue page
- User can view product cards
- User can search products
- User can sort products by price
- User can switch dark/light mode
- User can interact with buttons
- User can use the website on mobile devices

# Non-functional Requirements

- Semantic HTML5 must be used
- CSS must be stored in external files
- JavaScript must affect page functionality
- Responsive design must be implemented
- Website should support modern browsers
- User interface should be clear and modern
